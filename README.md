# ATE 6PRO v3.2

Это портативное Windows‑приложение для мониторинга рынка и автоторговли на OKX Spot.

## Главная идея v3
- **Только одна стратегия:** `StrategyV3` (детерминированные правила + per‑symbol пороги).
- В решении используется **Score** (0..1) — сила совпадения правил. Это не «вероятность».

## Быстрый старт
1) Собери EXE: `build_exe.bat`
2) Запусти: `dist\ATE_6PRO\ATE_6PRO.exe`
3) Для проверок из исходников: `pytest -q`

## Настройка порогов по каждой монете
См. `tools/generate_thresholds_from_dist.py` и документ `docs/STRATEGY_V3_DESCRIPTION_RU.md`.

## Безопасность
- По умолчанию автоторговля выключена.
- Для реальной торговли нужны OKX API ключи (environment variables).
